# المرقب الذهبي - لوحة التحكم

لوحة تحكم عقارية عربية مبنية باستخدام:

- Vite
- React
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Supabase

## التشغيل

1. ثبّت الحزم:

```bash
npm install
```

2. أنشئ ملف `.env.local` وأضف:

```env
VITE_SUPABASE_URL=your-project-url
VITE_SUPABASE_ANON_KEY=your-anon-key
```

3. شغّل المشروع:

```bash
npm run dev
```

## المزايا

- تسجيل دخول وحماية صفحات
- لوحة رئيسية بإحصائيات
- إدارة الطلبات مع لوحة تفاصيل
- إدارة العقارات والعملاء
- رسائل واتساب وقوالب
- إعدادات المكتب والمستخدمين

عند غياب متغيرات البيئة، تعمل الواجهة ببيانات تجريبية لتسهيل المعاينة.
